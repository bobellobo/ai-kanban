# ai-kanban
***Tech interview preparation***

A collaborative, real-time project management dashboard SaaS where a user can upload a product brief, and an AI agent breaks it down into actionable tasks.
The user and the team can then interact with the AI agent to refine the tasks, assign them, and track progress on a kanban board.

Built with Next.js and NestJS, this project serves as a comprehensive example of a full-stack application
